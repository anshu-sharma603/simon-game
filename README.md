## 📖 About

**Simon** is a digital recreation of the iconic memory game. The game challenges players to repeat a growing sequence of colored button flashes. Each round adds one more step to the sequence — how far can you go?

---

## 🎯 How to Play

1. Press **Start** to begin the game.
2. Watch the sequence of colored buttons that light up.
3. Repeat the sequence by clicking the buttons **in the same order**.
4. Each successful round adds one more step to the sequence.
5. One wrong click and it's **Game Over** — try to beat your high score!

---

## ✨ Features

- 🔴🟢🔵🟡 Four-color button pad with sound & light feedback
- 📈 Progressive difficulty — sequence grows with each level
- 🔊 Unique tones for each color button
- 💥 Game Over animation on wrong input
- 🏆 Score tracking to challenge yourself
- 📱 Responsive design — works on desktop and mobile
